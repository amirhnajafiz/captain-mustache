Dockerize your Golang applications under *Mustache*. 

A fast, safe, dynamic way to create dockerfiles for you golang applications.

## How to use?
Clone the repository:
```shell
git clone https://github.com/amirhnajafiz/mustache.git
```

Start script:
```shell
./bin/mustache
```
